THEBBS.PHP
==========

(unmaintained)

本スクリプトは, sugoiyo72氏がオカメ仮面氏制作の掲示板スクリプトTHEBBS.CGIを改造したスクリプト, THEBBS.CGI（改）2.6.2をPHPに書き直したものです.
基本的な使用方法は[THEBBS.CGI（改）2.6.2](readme/readme.html)の説明を見てください.

本スクリプトはmod_phpを前提に制作されています.
CGIでPHPを動かしている場合, 正常に動作しない可能性があります.

本スクリプトはTHEBBS.CGI（改）2Xシリーズとログに互換性がありません...
もとい現行スレッド一覧ログ「t」および終了済みスレッド一覧ログ「e」内の
00000000<a href="thebbs.cgi?・・・・・・・ という部分を
00000000<a href="thebbs.php?・・・・・・・ というように, cgiをphpに書き直すことで再利用可能です.

その他の諸注意などは, THEBBS.PHP内に記述しましたので, そちらをご覧ください.

---

本スクリプトは2006年に作成され, その後メンテナンスされていません.
ご利用に当たっては十分ご注意下さい.
