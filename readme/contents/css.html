CSS 簡易解説書

【 sugoiyo260.css 】


sugoiyo260.cssではそれぞれ、以下の項目に対して指定をしています。実際のファイルでの記述順とは並ぶ順序が異なっています。

全体的なもの
・全体の背景色
body{background-color:white}

・全体のリンク色等
a{COLOR:blue;font-size:96%;padding:2px;}
:hover {color:blue;text-decoration:underline;}

・タイトル帯部分の背景色・文字色・帯の幅・フォントサイズ・余白
.ti{background-color:dimgray;color:white;width:100%;font-size:large;padding:3px}

・削除済み記事のタイトル帯部分の背景色・文字色・その他
.ti2{background-color:dimgray;color:white;width:100%;font-size:14px;padding:3px}

・スクリプト名称表示のフォント・サイズ・文字色
.cp{position:absolute;right:2px;font-size:10px}
.cp a{COLOR:333333;font-family: 'verdana,chicago,"ＭＳ Ｐゴシック",Osaka';text-decoration:none;}


インデックス画面（スレッド一覧）

・サブタイトル部分の枠線・余白・フォントサイズ
.box{border:1px solid;margin:10px,30px;padding:5px;font-size:75%}

・インデックス（スレッドの一覧画面）でのHNの余白
.index b{padding:0,3}

・インデックスでのスレッドタイトルのフォントサイズ
.index dd{font-size:80%}

・インデックスでのスレッド情報（レス数,HN,日付）のフォントサイズや余白
.index dt{margin-top:5;text-indent:1em;font-size:96%}


スレッド表示画面

・admin暗号化文字列の色とフォントサイズ
i{color:red;font-size:smaller;}

・一般暗号化文字列の色とフォントサイズ
.thread em{color:gray;font-size:smaller;}

・削除記事の色とフォントサイズ
.thread small{color:gray}

・IPから生成する文字列の色や、スレッド番号やレス番号、スレッドタイトルのリンク色等
#IP{color:#eeeeee;text-decoration:none;font-size:smaller}
#IP:hover {color:#aaaaaa;text-decoration:none;}
#num{color:#000000;text-decoration:none;}
#num:hover {color:red;text-decoration:underline;}
#ti{color:white;text-decoration:none;}
#ti:hover {color:white;text-decoration:underline;}

・スレッド画面でのHNのフォントサイズ、下線
.thread b{text-decoration:underline;font-size:110%}

・スレッド画面での投稿情報（レス数,HN,日付）のフォントサイズや余白<・dt>
.thread dt{margin:15px,0,5px,0;font-size:92%}

・スレッド画面での投稿記事のフォントサイズや改行
.thread dd{word-wrap:break-word;font-size:90%;line-height:120%;}

※機能していません・・。
.thread input{border:1px solid}

※機能していません・・。
.thread textarea{border:1px solid;filter:}

・スレッド画面右上のインデックスへ戻るリンク部分のフォントサイズ、右詰
.mi{font-size:smaller;text-align:right}

・終了スレッドのメッセージ文字色
.end{color:red}

・終了スレッドのメッセージ文字色
.sma{font-size:80%;padding:10px}


その他

250より追加されたユーザー用文字列の文字色を設定
#uc{color:blue}
260より追加されたPaiNタグ内の表示を設定
.PaiN {background-color:#efefef;padding-top:15px;padding-bottom:15px;padding-left:7px;padding-right:7px;border:1px dimgray dotted}
.PaiN a{color:black;text-decoration:none;}
.PaiN:hover {color:black;text-decoration:none;}
